# OSTRIE — преміальний барбершоп

Односторінковий production-ready сайт барбершопу **OSTRIE** (вістря / лезо).
Брутальна, але вишукана темна естетика для чоловіків, які цінують точність і характер.

> **Тег-лайн:** На вістрі стилю.

## Стек
Чистий **HTML + CSS + JS** в одному файлі (`index.html`). Без збірки, без залежностей,
тільки шрифти Google Fonts. Відкривається подвійним кліком.

## Дизайн
- **Стиль:** dark editorial / modern brutalism з преміальним відтінком.
- **Палітра:** вугільний `#0c0c0d` · off-white `#ece7df` · латунний акцент `#c9742d`.
- **Шрифти:** Anton (display, UPPERCASE) + Manrope (body).

## Секції
Hero · Послуги та ціни · Майстри · Галерея робіт · Про нас · Відгуки · Онлайн-запис · Контакти.

## Можливості
- Sticky-навігація з якорями + мобільне меню.
- Scroll-reveal (IntersectionObserver), smooth-scroll, hover-мікроанімації.
- Форма запису з валідацією та екраном успіху.
- Mobile-first адаптив, доступність (focus, aria, `prefers-reduced-motion`).

## Запуск
Відкрити `index.html` у браузері. Для локального серверу:
```bash
npx serve .
```
